## 2. 캡처 체크포인트

- 실제 데이터가 보이는 화면을 우선 사용
- 상태/결과 텍스트가 읽히도록 폰트 크기 확인
- 핵심 가치(두뇌 훈련, 스트릭, 빠른 피드백)가 순서대로 전달되게 배치
- 동일한 테마/톤으로 세트 통일

## 3. 이미지 사양

- 권장 해상도: `1080 x 1920` (9:16)
- 형식: PNG 또는 JPG
- 권장 장수: 8장
- 파일 용량: 이미지당 8MB 이하
